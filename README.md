# 📊 BMart Store Data Analysis (Excel Project)

## 📖 Overview

This project focuses on analyzing retail store data using Microsoft Excel. The objective is to uncover meaningful insights related to sales performance, customer behavior, and business growth to support better decision-making.

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Charts & Dashboard

---

## 📊 Project Features

* Interactive dashboard for sales tracking
* Category-wise and product-wise analysis
* Region-wise performance comparison
* Monthly and yearly sales trends

---

## 🔍 Key Insights

* Identified top-performing product categories contributing the highest revenue
* Found peak sales periods based on monthly trends
* Analyzed regional performance to highlight high and low sales areas
* Detected underperforming products needing attention
* Evaluated overall sales and profit patterns

---

## 📁 Dataset Information

* Dataset Type: BigMart Store Data
* Total Records :30000+ Rows
* Fields Included: Sales, Profit, Category, Region, Date, Product Name

---

## 🚀 How to Use

1. Download the Excel file from this repository
2. Open it using Microsoft Excel
3. Explore the dashboard and pivot tables for insights

---

## 📌 Conclusion

This project demonstrates how Excel can be effectively used for data analysis and visualization. It highlights key business insights that can help improve sales strategies and overall performance.
